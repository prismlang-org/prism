Prism.errors error index
Error

STD::E001

Mismatched types

code E001/ED0x7374723E696E74
url https://prismlang-org.github.io/prism/errors/e001/ed0x7374723E696E74
description
Raised when a value of one type is used where a different, incompatible type is expected and the compiler cannot safely coerce it. If a safe coercion path exists — such as passing an int where a float is expected — the compiler emits W0x000001 instead and coerces automatically. STD::E001 is only raised when coercion is genuinely unsafe or impossible.
example
example.prism — triggers STD::E001 
fn add(a: int, b: int): int begin:
    add::return << a + b;
end;

add("hello", 5); -- STD::E001: str passed where int expected
fix
example.prism — corrected 
fn add(a: int, b: int): int begin:
    add::return << a + b;
end;

add(10, 5); -- correct: both arguments are int
notes
If you intended to convert a value between types, use an explicit cast instead. Note that not all casts are permitted — see STD::E002 for invalid cast errors.
